Altered implementation of Dijkstra's algorithm that ignores specified nodes and edges

Calculates the shortest driving path from a source node to a destination node, without passing by any of the nodes in the forbidden set or any of the edges in the avoidSegments set

Note
Time Complexity: O((V + E) log V)
